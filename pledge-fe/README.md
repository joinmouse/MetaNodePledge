# Market Pool

Market Pool 是固定利率借贷市场，连接出借方和借款方。

## 核心功能

- **出借方（Lender）**：提供稳定币（BUSD/USDT/PLGR），锁定固定利率收益
- **借款方（Borrower）**：抵押加密资产（BTCB/ETH等），借出稳定币使用

## 池子分类

- **BUSD池**：主流稳定币借贷池
- **USDT池**：预留功能（即将上线）
- **PLGR池**：协议代币池，提供更高收益激励

## 关键参数

- **Fixed Rate**：固定年化利率
- **Margin Ratio**：清算阈值，控制风险
- **Settlement Date**：结算时间
- **Length**：借贷周期
- **Interest**：利息
- **Liquidation Price**：清算价格
- **Liquidation Ratio**：清算阈值
